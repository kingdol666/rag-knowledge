## Problem

Neo4j Docker 容器未启动，端口 7687 未监听，导致所有 `/api/v1/graph/*` 接口返回 500 Internal Server Error：

- `/api/v1/graph/search?keyword=test` — 500
- `/api/v1/graph/stats` — 500
- `/api/v1/graph/neighbors` — 500
- 两阶段检索中的图谱邻居扩展（`enable_graph_expansion=true`）也无法使用

## Solution

1. 确保 Docker Desktop 在运行
2. 执行 `docker compose up -d` 启动 Neo4j 容器
3. 验证连接：`curl http://localhost:7687` 应返回 Neo4j 握手响应
4. 检查 `config.yml` 中的 `graph.password` 配置与 Docker Compose 中的 `NEO4J_PASSWORD` 一致
